# 五文件需求覆盖详细分析

## 一、文件清单

1. **涌益周度数据.xlsx** (`2026.1.16-2026.1.22涌益咨询 周度数据.xlsx`)
   - Sheet数量：65个
   - 数据内容：周度价格、体重、屠宰、利润、冻品、月度存栏等
   - 数据源：YONGYI

2. **涌益日度数据.xlsx** (`2026年2月2日涌益咨询日度数据.xlsx`)
   - Sheet数量：8个
   - 数据内容：日度价格、屠宰量、价差、交割地市等
   - 数据源：YONGYI

3. **lh_ftr.xlsx** - 期货日历史行情数据
   - 数据源：DCE（大连商品交易所）
   - 数据内容：期货合约日度行情（开盘价、收盘价、结算价、成交量、持仓量等）

4. **lh_opt.xlsx** - 期权日历史行情数据
   - 数据源：DCE（大连商品交易所）
   - 数据内容：期权合约日度行情（含IV隐含波动率、Delta等）

5. **钢联自动更新模板.xlsx** (`1、价格：钢联自动更新模板.xlsx`)
   - Sheet数量：7个
   - 数据内容：钢联价格数据（分省区猪价、集团企业出栏价、区域价差、毛白价差等）
   - 数据源：GANGLIAN（上海钢联）
