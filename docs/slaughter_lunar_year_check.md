# 日度屠宰量农历年区间核对（2021～2025）

## 数据源（A3 日度屠宰量）

- **原始 Excel**：涌益咨询日度数据，Sheet「价格+宰量」。
- **最新数据目录**：`docs/0306_生猪/生猪`。该目录下含涌益日度 xlsx 时，导入脚本会优先从此目录匹配（文件名可含「涌益」「日度」，如「1 涌益咨询日度数据.xlsx」）。
- **导入命令**（项目根目录）：
  ```bash
  python import_data.py "docs/0306_生猪/生猪"
  ```
  或 Windows 绝对路径：`python import_data.py "D:\Workspace\hogprice-insight\docs\0306_生猪\生猪"`。

## 缺口说明

- **缺尾**：库内最晚日期早于该年「腊月二十八」阳历日，即年末几天无数据（多为 2～3 天）。
- **缺首**：仅 2024 年库内最早为 2024-02-18，比正月初八 2024-02-17 少 1 天。

## 图表是否正确

- 接口已按「农历年 = 正月初八～腊月二十八 的阳历区间」查询，图表展示的即是该区间内**库内实际有的**数据。
- 因此图表逻辑正确；若某年曲线未到腊月二十八或未从正月初八起，是因为**数据源（涌益 Excel）在该区间内本身缺这些日期**，而非查询或前端画图错误。
- 若需各年“完整”到腊月二十八，需在数据源/导入中补全对应日期的记录。
