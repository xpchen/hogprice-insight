## 四、4.2涌益底稿 可用数据

- **表头**：第 1 行为大类（大猪存栏、能繁母猪、饲料销量环比等），第 2 行为子表头（全国、东北、华北等，或猪料销量、后备母猪料、其他母猪料等）。
- **数据**：从第 3 行开始。

与 E2 表格1 相关的列（列索引从 0 起，对应 API 取数）：

| 列索引 | 列名 | 行1 | 行2 | 当前 API 使用 |
|--------|------|-----|-----|----------------|
| 0 | A | (日期) | - | ✅ 日期 |
| 1 | B | 大猪存栏-2020年5月* | 全国 | ✅ 大猪存栏环比 → hog_inventory_yongyi |
| 6 | G | 能繁母猪-2020年2月* | 全国 | ✅ 能繁存栏环比 → breeding_inventory_yongyi |
| 15 | P | 饲料销量环比 | 猪料销量 | ✅ 猪料销量环比 → hog_feed_yongyi（育肥猪饲料） |
| 16 | Q | (同上块) | **后备母猪料** | 可作能繁母猪饲料环比 → breeding_feed_yongyi |
| 17 | R | (同上块) | **其他母猪料** | 可选：与 Q 合并或单独使用 |

结论：

- **能繁母猪饲料环比-涌益**：4.2涌益底稿中**没有**名为「能繁母猪饲料环比」的列，但有：
  - **Q 列（索引 16）**：后备母猪料  
  - **R 列（索引 17）**：其他母猪料  
  业务上「能繁母猪饲料」常对应后备母猪料（或后备+其他母猪料）。可用 **Q 列** 作为「能繁母猪饲料环比-涌益」的数据来源并接入 API；若需包含「其他母猪料」，再考虑 R 或 Q/R 组合。

## 五、建议

1. **淘汰母猪屠宰**：钢联底稿、4.2涌益底稿均无此数据，维持从「涌益样本」或涌益周度「月度-淘汰母猪屠宰厂宰杀量」取数。
2. **能繁母猪饲料环比-涌益**：在 `_extract_yongyi_data()` 中增加对 4.2涌益底稿 **Q 列（索引 16）** 的读取，映射为 `breeding_feed_yongyi`；若业务确认为「后备+其他母猪料」，再增加 R 列或 Q/R 合并逻辑。
